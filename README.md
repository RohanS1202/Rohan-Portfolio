# Rohan Sagvekar — Portfolio

Personal portfolio website for Rohan Sagvekar, Data Scientist & ML Engineer.

## Tech Stack

React, TypeScript, GSAP, Three.js, WebGL, HTML, CSS, JavaScript

## Getting Started

```bash
npm install
npm run dev
```

## Note

This project uses GSAP trial plugins. For production hosting, replace with GSAP Club plugins.
Check out: https://gsap.com/docs/v3/Installation/
